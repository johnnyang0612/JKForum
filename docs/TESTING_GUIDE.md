# JKForum — 客戶測試指南（普通使用者版）

> **這份指南讓你不用懂技術，照著點就能完整測試所有功能。**
> 不需要打 Email、不需要記密碼、不需要進後台跑 SQL — 全部靠登入頁的「**一鍵登入**」按鈕切換身份。

- 線上測試網址：https://jkforum.vercel.app
- 本機測試網址：http://localhost:3000
- 最後更新：2026-05-05

---

## 一、開始之前（30 秒搞懂）

### 1. 打開網站
進到 https://jkforum.vercel.app/login

### 2. 看到一張黃色「🧪 測試用一鍵登入」卡片
裡面有 **6 顆按鈕**，對應 6 種使用者身份：

| 按鈕 | 身份說明 | 你會看到什麼 |
|------|---------|-------------|
| 🛡️ **管理員** | 站方最高權限 | 進站後右下角有「管理員後台」入口、可去 `/admin` |
| 🛠️ **版主** | 管特定版區的小管理員 | 在自己負責的版可置頂/刪文，其他版同一般會員 |
| ✍️ **駐站編輯** | 內容編輯 | 可寫官方文章、有「編輯」徽章，但不能管後台 |
| 💎 **VIP 會員** | 付費會員 | 頭像旁有 VIP 徽章、可看 VIP 限定內容 |
| 👑 **資深會員** | 高等級老會員 | 可進高權限區、發文無限、雙倍簽到 |
| 🌱 **新手會員** | 剛註冊的菜鳥 | 很多區進不去、發文有限、簽到普通 |

> 想測「**訪客**」（沒登入）？最簡單：開「無痕視窗」直接逛，**不要按任何登入按鈕**。

### 3. 切換身份
按一下 → 會自動登入 → 跳到首頁。
要換身份？右上角頭像 → **登出** → 回登入頁再按別的按鈕。

## 三、跨身份共通測試

### 響應式（手機 / 平板 / 桌面）
1. 桌面：直接用瀏覽器
2. 平板：DevTools → 切「iPad」（768×1024）
3. 手機：DevTools → 切「iPhone 14 Pro」（390×844）

預期：
- **手機**：底部出現 5 顆固定 nav（首頁/聊天/發文/遊戲/我的）、左側選單收進漢堡
- **平板**：左側選單變窄
- **桌面**：3 欄佈局（左選單 / 內容 / 右側資訊）

### PWA 安裝
1. 用 Chrome 桌面 → 網址列右側 ⊕ → 「安裝 JKForum」
2. 用 Chrome 手機 → 三點選單 → 「加到主畫面」
3. 開啟 → 沒有瀏覽器列、像原生 App

### 推播通知
1. 進 `/settings` → 通知 → 啟用 Web Push
2. 同意瀏覽器權限
3. 請另一個管理員到 `/admin/push` 廣播一則
4. 你的瀏覽器會跳通知（即使關閉分頁）

### 深色 / 淺色
- 右上角 ☀ / 🌙 圖示切換
- 預期：所有顏色平滑過渡

### 搜尋
- 頂部放大鏡 → 打「美食」
- 預期：5 個分頁（全部 / 文章 / 用戶 / 看板 / 日誌）
